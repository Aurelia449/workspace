常见问题

LUMINA 和直接调用 OpenAI / Claude API 有什么区别？+
LUMINA 不是转发代理。我们提供自研推理引擎、私有化部署能力、自定义微调、多模态统一 API 和企业级 SLA。对于有定制化、合规、成本控制需求的企业，LUMINA 是真正的基础设施层选择。
数据安全和隐私如何保障？+
所有数据传输均使用 TLS 1.3 加密，推理过程不会存储用户数据，通过 SOC2 Type II 认证。企业版支持完全私有化部署，数据物理隔离，满足金融、医疗等强合规场景。
接入需要多少技术成本？+
标准 REST API，5 分钟即可完成接入。我们提供 Python、Node.js、Go、Java 等主流语言 SDK，以及 LangChain、LlamaIndex 等主流框架的原生集成。现有系统无需大幅改造。
如何计费？有没有隐藏费用？+
按 Token 用量计费，价格透明，官网实时可查。入门版永久免费，专业版每月固定费率含 10 亿 Token，超出部分按量计费。没有任何隐藏费用，账单精确到每一个 Token。
支持哪些模型？能用自己的模型吗？+
支持 200+ 主流开源模型（LLaMA、Mistral、Qwen、Gemma 等），同时支持上传私有模型或通过 Fine-tune 平台定制专属模型，部署与调用完全自主可控。
