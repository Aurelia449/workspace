SQL 建表

在 Supabase SQL Editor 中运行以下脚本

① 创建 company_news 表

▶
create table if not exists company_news ( id uuid primary key default gen_random_uuid(), title text not null, summary text, content text, category text default 'announcement', status text default 'draft', cover_url text, author text default '管理员', tags text[] default '{}', created_at timestamptz default now(), updated_at timestamptz default now() );

② 创建 ai_features 表

▶
create table if not exists ai_features ( id uuid primary key default gen_random_uuid(), name text not null, description text, category text default 'core', status text default 'active', icon text, sort_order int default 0, highlights text[] default '{}', docs_url text, created_at timestamptz default now(), updated_at timestamptz default now() );

③ Row Level Security（开发阶段允许全部操作）

▶
alter table company_news enable row level security; alter table ai_features enable row level security; create policy "allow all" on company_news for all using (true) with check (true); create policy "allow all" on ai_features for all using (true) with check (true);
